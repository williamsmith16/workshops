# Digital Art & Animation - 2 Hour Workshop

**Date:** February 26, 2026
**Audience:** Beginners (ages ~12-14) \
**Theme:** _"Designing your own character sprites and bringing them to life in a game engine"_

- [Setup Guide]({{ site.baseurl }}/character-sprite-animation/HOW_TO_SETUP)
- [Starter Project Zip]({{ site.baseurl }}/character-sprite-animation/godot-character-boilerplate.zip)

## Workshop Overview

In this workshop, students will customize character sprite animations using
[Piskel](https://www.piskelapp.com/) and import them into a ready-made Godot
platformer. Students will learn about sprite sheets, pixel art, and animation
frames while creating their own unique playable character.

### What You'll Learn

- What **sprite sheets** are and how animations are built from frames
- How to edit pixel art using the **Piskel** web editor
- How to import custom art into the **Godot** game engine
- How game engines use image files to animate characters

### Tools Used

- [Piskel](https://www.piskelapp.com/) — free online pixel art editor
- [Godot Web Editor](https://editor.godotengine.org) — free game engine (browser version)
- [Deymoon Character Template](https://deymoon.itch.io/character-template) — base character sprites
